# Dovira

Dovira - French video editing agency website built with Next.js 15.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Next.js 15 with App Router
- Tailwind CSS with Shadcn/UI components
- Motion animations (previously Framer Motion)
- Responsive design with premium dark theme
- Video portfolio carousel
- Contact forms and pricing sections
